# 🌿 Ouro da Terra - Landing Page Premium

Uma landing page sofisticada e responsiva desenvolvida para a marca **Ouro da Terra**, especializada em temperos artesanais do Nordeste brasileiro. O projeto foca em uma estética "Dark & Gold" para transmitir exclusividade, tradição e pureza.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando uma abordagem moderna de desenvolvimento rápido:

* **HTML5 & CSS3 Semântico**: Estrutura focada em SEO e acessibilidade.
* **Tailwind CSS**: Framework utilitário para estilização rápida e design responsivo.
* **AOS.js (Animate On Scroll)**: Biblioteca para animações de entrada que elevam a experiência do usuário.
* **Google Fonts**: Utilização das famílias *Playfair Display* (Serifada clássica) e *Lora* (Elegância rústica).
* **IA Generativa (Gemini)**: Utilizada como copiloto para estruturação de código, copywriting e curadoria de assets.

## 🛠️ Funcionalidades

- [x] **Design Responsivo**: Adaptável para Mobile, Tablet e Desktop.
- [x] **Animações Fluidas**: Elementos que surgem conforme o scroll do usuário.
- [x] **Performance Otimizada**: Carregamento rápido via CDNs globais.
- [x] **Identidade Visual Temática**: Paleta de cores baseada em tons terrosos e dourado.
